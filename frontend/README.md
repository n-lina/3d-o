Environment setup: 
1. Clone this repository 
2. Download Git, Node.js and NPM 
3. cd frontend 
4. npm install 
5. npm run start 

To build and deploy: 
1. cd frontend 
2. npm run build 
3. push changes -- auto-deployed with Heroku

To run build version locally: 
1. cd 3d-o 
2. npm install
3. npm run start
